PHP-Facebook-Connect-Login by amanelis

A light weight PHP Facebook connect log in script to handle users and interact with database.

Using these files, you are able to use Facebook connect to log users into your site, store Facebook user id in database, check on each log in whether they are new or returning user. Also included are several versions that use only the java script library to connect.

Dependencies

The only dependency, which I have included for you are the facebook-client library, and the xd_receiver.htm file. The only thing you need is your Facebook's application key and secret.

Install

Copy all files into directory where you will be hosting your application. Follow all instructions on README file as to where you need to change your API key and secret.
